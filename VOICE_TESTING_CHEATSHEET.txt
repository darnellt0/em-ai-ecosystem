THE 6 ENDPOINTS

  1. Block Focus Time
     POST /api/voice/scheduler/block
     Input:  {"minutes": 45, "founder": "shria"}
     Output: "Blocked 45 minutes for focus..."

  2. Confirm Meeting
     POST /api/voice/scheduler/confirm
     Input:  {"title": "Team Sync", "startAtISO": "...", "durationMinutes": 60}
     Output: "Added Team Sync to calendar..."

  3. Reschedule Meeting
     POST /api/voice/scheduler/reschedule
     Input:  {"eventId": "evt-123", "newStartAtISO": "..."}
     Output: "Rescheduled event evt-123..."

  4. Start Meditation
     POST /api/voice/coach/pause
     Input:  {"style": "grounding", "seconds": 60}
     Output: "Starting a 60s grounding meditation..."

  5. Log Task Complete
     POST /api/voice/support/log-complete
     Input:  {"taskId": "task-789"}
     Output: "Marked task task-789 as complete..."

  6. Create Follow-up
     POST /api/voice/support/follow-up
     Input:  {"subject": "Follow up on Q4"}
     Output: "Created follow-up: Follow up on Q4..."

TROUBLESHOOTING

  Q: "ffplay: command not found"
  A: brew install ffmpeg

  Q: "ELEVENLABS_API_KEY not set"
  A: export ELEVENLABS_API_KEY="sk-your-actual-key"

  Q: No sound but no error
  A: Try: open response.mp3 (or ffplay response.mp3)

  Q: API returns 401
  A: Check API is running: docker-compose ps

  Q: API returns 400
  A: Check your request parameters
