## Features in Detail

### Offline Mode

Work without internet, sync when you're back online.

**How It Works**:
1. Voice commands are recorded locally
2. They're queued for sending when online
3. Automatic sync happens in background
4. You're notified when sync completes

**Limitations**:
- Voice transcription requires internet
- Some features need online connection
- Queue holds up to 100 items

**Checking Sync Status**:
1. Go to Profile tab
2. View "Sync Status" section
3. See: Online/Offline, Last Sync Time, Pending Items

**Manual Sync**:
1. Pull down on any screen to refresh

## Troubleshooting

### Voice Command Not Working

**Issue**: Microphone icon grayed out or no response

**Solutions**:
1. Check microphone permissions:
   - iOS: Settings ‚Üí Privacy ‚Üí Microphone ‚Üí Enable for app
   - Android: Settings ‚Üí Apps ‚Üí Permissions ‚Üí Microphone ‚Üí Allow

2. Restart the app

3. Check internet connection

4. Update app to latest version

### App Crashes

**Issue**: App closes unexpectedly

**Solutions**:
1. Force close and reopen
2. Clear app cache (Settings ‚Üí Clear Cache)
3. Update to latest version
4. Reinstall if persistent
5. Contact support with details

### Sync Issues

**Issue**: Commands not syncing

**Solutions**:
1. Check internet connection
2. Go to Profile ‚Üí View sync status
3. Manual sync: Pull to refresh
4. Check API server status
5. Logout and login again

### Login Problems

**Issue**: Can't login or "Invalid credentials"

**Solutions**:
1. Verify email and password
2. Check internet connection
3. Reset password (coming soon)
4. Contact support if account locked

### Offline Mode Not Working

**Issue**: Commands not queuing offline

**Solutions**:
1. Settings ‚Üí Enable offline mode
2. Grant storage permissions
3. Check available storage space
4. Clear cache if full

## Privacy & Security

### Data Storage

- **Local Storage**: Voice recordings, user preferences, offline queue

### Permissions

**Required**:
